filter

Метод arr.filter(callback[, thisArg]) використовується для фільтрації масиву через функцію.

Він створює новий масив, в який ввійдуть тільки ті елементи arr, для яких виклик callback(item, i, arr)поверне true.

Наприклад:

var positiveArr = arr.filter(function(number) { return number > 0; });
text text

alert( positiveArr ); // 1,2,3
